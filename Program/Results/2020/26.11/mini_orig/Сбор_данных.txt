Приложен архив с тремя файлами:
 mini.CPP - исходный код программы с комментариями,
 MINI.EXE - скомпиленная программа,
 1.CSV - пример сформированного файла с данными.

 После запуска программа выводит сообщение о готовности и по нажатию
 любой клавиши переходит в режим генерации задающих воздействий и
 сбора данных. В качестве задающего воздействия сейчас взята синусоида.
 Данные записываются с частотой примерно 291 Гц, максимальное
 количество тактов - 16000, то есть примерно 50 секунд работы.
 По истечении этого времени либо нажатии клавиши esc работа
 останавливается и выводится запрос имени файла для сохранения данных.
 Даные сохраняются в текстовом виде в формате csv, который может быть
 открыт, например, в ms excel, поэтому имя файла желательно вводить в
 виде *.csv.

 Для компиляции исходного кода в borland c++ 3.1 надо в настройках
 компилятора (options -> compiler -> advanced code generation)
 включить настройку Automatic far data. В настройках
 компилятора (options -> compiler -> code generation) желательно
 выбрать Large в поле Model.
